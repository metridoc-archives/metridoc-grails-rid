(Quick Reference)

configure-for-job-only

Usage

grails configure-for-job-only [job name]

Description

Configures an application / plugin as a job only project. Grails creates a lot of folders and files by default, a lot of which are unnecessary when you only need to create jobs. If a job name is not provided the user will be prompted to add one.
